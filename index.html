Session Anomaly Detector

Flags rapid-answer bursts and unusual inactivity gaps in student quiz session logs to support academic integrity review.

🔧 Install the Extension One-time setup

  1. Download the extension file Click the button below to download canvas-nq-anomaly.crx to your computer.
    ⬇ Download Extension
  2. Open Chrome's extension manager In a new Chrome tab, type chrome://extensions in the address bar and press Enter.
  3. Enable Developer mode Find the Developer mode toggle in the top-right corner of the page and switch it on.
    Why is this needed? Chrome requires this one-time step to install extensions from outside the Chrome Web Store. You can turn it off again after installing if you prefer.
  4. Drag the file into Chrome Drag the downloaded canvas-nq-anomaly.crx file from your Downloads folder into the chrome://extensions tab. A prompt will appear — click Add extension.
  5. Done! The magnifying glass icon (🔍) will appear in your Chrome toolbar. If you don't see it, click the puzzle-piece icon (🧩) to find and pin it.

▶ How to Run an Analysis Each exam

  1. Open the quiz in Canvas Go to the course in Canvas → Assignments → click the quiz you want to review.
  2. Navigate to the Moderate tab Inside the quiz editor, click the Moderate tab at the top. Wait for the student list to fully load.
  3. Click the extension icon Click the 🔍 icon in the Chrome toolbar. A small popup appears with a Run Anomaly Check button.
  4. Review the settings and run A configuration window appears. The defaults work well for most exams — adjust the thresholds if needed, then click ▶ Run Analysis.
    For a 2-hour exam you may want to increase the Gap threshold to 15–20 minutes to avoid flagging normal thinking pauses.
  5. Wait for the report The tool fetches each student's session log in the background (this can take 30–60 seconds for a large class). A printable report opens automatically when complete.

🔄 Updates

The extension updates itself automatically. When a new version is released, Chrome will install it in the background — you don't need to do anything.
