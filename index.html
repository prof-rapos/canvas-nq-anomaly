Session Anomaly Detector

Identifies and flags multiple sessions, rapid answers, and long pauses in quizzes, with logs available to review.

🔧 Install the Extension One-time setup

  1. Download and extract the extension Click the button below to download the zip file. Extract it somewhere safe on your computer.
    ⬇ Download Extension (.zip)
  2. Open Chrome's extension manager In a new Chrome tab, type chrome://extensions (or click to copy then paste) in the address bar and press Enter.
  3. Enable Developer mode Find the Developer mode toggle in the top-right corner of the page and switch it on.
    Why is this needed? Chrome requires this one-time step to install extensions from outside the Chrome Web Store. You can leave it on — it doesn't affect anything else.
  4. Click "Load unpacked" A button labelled Load unpacked will appear at the top-left. Click it, then navigate to and select the canvas-nq-anomaly folder you extracted in step 1.
  5. Done! The icon appears in your Chrome toolbar. If you don't see it, click the puzzle-piece 🧩 icon to find and pin it.

▶ How to Run an Analysis Each exam

  1. Open the quiz in Canvas Go to the course in Canvas → Assignments → click the quiz you want to review.
  2. Navigate to the Moderate tab Inside the quiz editor, click the Moderate tab at the top. Wait for the student list to fully load.
  3. Click the extension icon Click the icon in the Chrome toolbar. A small popup appears — click Run Anomaly Check.
  4. Set the sensitivity and run A window appears asking how many questions answered within 30 seconds should be flagged. Drag the slider to your preferred sensitivity, then click ▶ Run Analysis.
    The lower the value, the more likely you are to flag large amounts of students.
  5. Wait for the report The tool fetches each student's session log in the background (30–60 seconds for a large class). A report opens automatically when complete, where you can view individual logs, or export all flagged logs.

🔄 Updating to a New Version

  1. Download and extract the new zip Download the latest version above and extract it, replacing the old canvas-nq-anomaly folder.
  2. Reload the extension Go to chrome://extensions, find the extension, and click the ↻ reload icon. That's it.
